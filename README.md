# Jekyll Personal Blog Template


You can host websites in GitHub using Github Pages feature.
It even supports a website creation scheme with only using markdown.
This technology is called Jekyll.

It takes a directory of .md files, and does the job of converting it into html files with themes and other blog features by itself.

If you want to host a small website, that will be under 50 blog entries, Jekyll is a basic and simple option.


If you are more ambitious about your blog,
you may also want to check another static site generator, Hugo (written in Go and faster, but more complicated).
